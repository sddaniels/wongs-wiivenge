Final Project Readme

Included Folders

  • client - Source code for the game client
  • darwiin - Source code and executable for the Wiimote driver
  • game_engine - Source code for the game engine
  • jogl mac osx - Java OpenGL bindings for Macintosh OSX
  • jogl windows - Java OpenGL bindings for Microsoft Windows
  • sockets - Source code for the client-server networking

How to Set Up Wong's Wiivenge

We developed the project using the free Eclipse IDE. Unfortunately we haven't been able to get the project to run outside of Eclipse becuase of problems setting the classpath and linking to the native libraries necessary to use OpenGL with Java. So if you would like to run the project, the best way is to download the program from Eclipse.org. No installation is necessary, and it should run as long as you have the correct version of Java from java.sun.com.

Once Eclipse is unzipped, follow these directions:

  1. Run Eclipse.
  2. When the popup dialog asks where you would like to place your workspace, select any directory you wish to work from.
  3. Go to File -> New and select Project.
  4. Select "Java Project" and click Next.
  5. Name the project anything you like and click Finish.
  6. The project you created should appear in the Package Expolorer window on the left side of the screen. Right click on the project icon and select Import.
  7. Select "File System" and click Next.
  8. Click Browse on the right side of the dialog.
  9. Select the directory where you unzipped the contents of this final project.
  10. Click the + icon or the arrow next to the folder to expand the contents of the folder.
  11. Click on the check boxes next to "client", "game_engine", and "sockets". This will select all of the contents of these folders for import.
  12. Click Finish. The necessary source files should now appear under the project in the Package Explorer.
  13. Right click on the project icon again and select Properties.
  14. Click on "Java Build Path".
  15. Select the Libraries tab.
  16. Click the "Add External Jars" button.
  17. Again go to the directory where you unzipped the contents of this final project. If you're running Windows, go into the "jogl windows" folder. If you're running Mac OSX, go into the "jogl mac osx" folder. Select "jogl.jar" and click Open.
  18. Click the + icon or the arrow next to jogl.jar to expand its properties.
  19. Select "Native library location".
  20. Click the Edit button.
  21. Click the External Folder button.
  22. Go back to the directory you selected that contains jogl.jar and click Choose.
  23. Click the OK button.
  24. Click the OK button to close the project properties window.
  25. Go to the Run menu and select "Run...".
  26. Select "Java Application" and click the new button which looks like a page with a + symbol on the upper right.
  27. Give the configuration a name like "GameServer".
  28. Click the Search button under "Main class:".
  29. Double click GameServer.
  30. Click the Run button to run the game's server.
  31. Follow the same procedure again, but use the name GameClient and select GameClient instead of GameServer after clicking Search.
  32. You should now be able to run both the server and the client from the drop down menu of the Run button in the toolbar, which looks like a green circle with a white arrow in it.

How to Run the Game

  1. You need at least two computers with public IP addresses or connected on a LAN subnet to play the game.
  2. Start the game server on one of the computers. You may enter a new port for incoming connections or just hit enter to use the default. The game server should now wait for connections from clients.
  3. If you want to use a Wiimote with the game (Mac only), start the modified Darwiin program on each client computer. This is contained in the darwiin folder. Hold down the 1 and 2 buttons simultaneously to connect the remote. Note: A compatible Bluetooth adapter is required to connect a Wiimote.
  4. Start the game client on each computer. If the client is running on the same computer as the server, leave the game server address at the default. If it is running on another computer, type in the address of the computer running the server (the server should print this to output when it's started.
  5. Make sure the correct port is listed and the frame rate is set to 60.
  6. Click "Connect to Server".
  7. You should now be able play the game.

How to Play the Game

The objective is to defeat your opponent by draining their health to zero. The health is indicated by the green/red bars at the top of the screen. Each swing of the lightsaber drains your opponent's health if you are close enough to them.

Keyboard Input:

  • Move left and right: left and right arrow keys
  • Dodge back: down arrow key
  • Block: s key
  • Horizontal swing: d key
  • Vertical swing: f key

Wiimote Input:

    Move left and right: left and right d-pad buttons
  • Dodge back: down d-pad button
  • Block: up d-pad button
  • Horizontal swing: move the wiimote in a vertical swinging motion
  • Vertical swing: move the wiimote in a horizontal swinging motion
